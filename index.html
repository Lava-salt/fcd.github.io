Project FCD

What is Project FCD?

Project FCD means "Project Free Common Documents", which is an open source fully editable Google Docs document.

What is it used for?

You can put advertisement to the P-FCD document, so other people that look at that document will see that.
Or, you can write a message to there for one of your friends to see it. Be sure to not put personal information there.
Finally, you can make one page your personal homepage (but remember, anyone can delete that!) just like GeoCities.

How can I enter there?
